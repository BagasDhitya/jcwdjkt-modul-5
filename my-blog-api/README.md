# My Blog API

REST API untuk platform blogging yang dibangun dengan Node.js, TypeScript, dan Prisma ORM.

## Tech Stack

- **Runtime:** Node.js (v25+)
- **Language:** TypeScript
- **Framework/Library:** Express / Fastify (atau sesuaikan dengan framework utama kamu)
- **ORM:** Prisma
- **Database:** PostgreSQL / Supabase
- **Testing:** Jest & ts-jest
- **CI/CD:** GitHub Actions
- **Automation:** QStash / Upstash (untuk scheduling artikel)
